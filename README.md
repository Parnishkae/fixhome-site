# РемонтПро — сайт компании по ремонту

Современный сайт для компании ремонтных услуг. Чистый стек: HTML5 + CSS3 + Vanilla JS, без сборщиков и тяжёлых библиотек. Backend (опционально) — на PHP.

## 📁 Структура проекта

```
project/
├── index.html              # Главная страница
├── services.html           # Каталог услуг
├── about.html              # О компании
├── contacts.html           # Контакты
│
├── css/
│   └── style.css           # Все стили сайта
│
├── js/
│   └── app.js              # Логика (меню, формы, загрузка услуг)
│
├── data/
│   └── services.json       # Данные об услугах
│
├── images/                 # Фото услуг (положите свои сюда)
│   ├── service1.jpg
│   ├── service2.jpg
│   └── ...
│
└── admin/
    └── add-service.php     # Админка для добавления услуг
```

## 🚀 Как запустить локально

### Вариант 1 — Live Server (рекомендуется для разработки)
1. Откройте папку в VS Code.
2. Установите расширение **Live Server**.
3. Кликните правой кнопкой на `index.html` → **Open with Live Server**.

### Вариант 2 — встроенный сервер Python
```bash
cd project
python3 -m http.server 8000
```
Откройте `http://localhost:8000`.

### Вариант 3 — встроенный сервер PHP (нужен для админки)
```bash
cd project
php -S localhost:8000
```
Откройте `http://localhost:8000`.

> ⚠️ **Важно:** просто открывать `index.html` двойным кликом из проводника **нельзя** — браузер заблокирует загрузку `services.json` из-за CORS. Используйте локальный сервер.

## ✏️ Как добавить услугу

### Способ 1 — вручную через JSON
Откройте `data/services.json` и добавьте новый объект в массив:

```json
{
    "title": "Название услуги",
    "price": "от 500$",
    "description": "Описание услуги для клиентов.",
    "image": "images/my-service.jpg",
    "popular": true
}
```

- `popular: true` — услуга появится в блоке «Популярные услуги» на главной.
- `image` — можно оставить пустой строкой `""`, тогда покажется градиентный фон.

Сохраните файл — изменения видны сразу после перезагрузки страницы.

### Способ 2 — через PHP-админку
1. Запустите проект через PHP-сервер (см. выше).
2. Откройте `http://localhost:8000/admin/add-service.php`.
3. Заполните форму и нажмите «Добавить услугу».

## 🎨 Где менять контент

| Что | Где |
|---|---|
| Цвета и шрифты | `css/style.css` → блок `:root` |
| Логотип, телефоны, email | в каждом HTML-файле + footer |
| Преимущества, отзывы | в `index.html` |
| Этапы работ, ценности | в `about.html` |
| Услуги | `data/services.json` |
| WhatsApp/Telegram ссылки | поиск `wa.me` и `t.me` во всех HTML |

## 🛡️ Перед публикацией на боевой сервер

1. **Защитите админку** — добавьте Basic Auth (htpasswd) или сессионную авторизацию.
2. **Включите HTTPS** — обязательно для современных сайтов.
3. **Замените контакты** — телефоны, адрес, ссылки на мессенджеры.
4. **Подключите реальную отправку заявок** — в `js/app.js` функция `handleFormSubmit` сейчас только имитирует отправку (`setTimeout`). Замените на `fetch('/api/order.php', ...)`.
5. **Сжимайте изображения** — используйте формат WebP, размер ≤ 200 КБ для карточек.
6. **Минифицируйте CSS/JS** перед деплоем.

- Плавный скролл по якорям
